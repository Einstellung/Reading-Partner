记忆部分已于 2026-07-17 从北极星转正为 M8 落地：挂断静默蒸馏、开场主题快照、memory 工具、对话纠错、"看看 AI 记了什么"面板。设计依据与落地状态见 [02](../02-AI核心与memory设计.md) 第二部分。本文只留还没做的部分。

## 仍留在北极星的：主动联想

读这本时自然接上你在那本划过聊过的东西。人的联想在记忆里发生——现在 memory 有了，但跨书/跨主题联想的检索方向、触发时机、不打扰的呈现还没有体感，先用 M8 攒数据再回来做。

## 将来做时已知的事实

- 借自 HMS 的三条规矩已随 M8 进了 prompt 与蒸馏规则（主动回忆、写入端绝对日期、矛盾时保留演变），联想功能沿用即可。
- 检索第一版 BM25 已落地；长大后再评估混合检索（memsearch）与时间衰减。
- 高信号事件即时写入通道（划线、批注、⭐ 发生时直写 memory）M8 未做，联想要用到用户信号权重时先补这条。
- Memory Adapter 窄接口（retain/recall/listObservations/correct/rebuild）已就位，引擎可替换；联想只应写在 adapter 之上。
